Secant Method Solved Example

Example:

Compute two iterations for the function f(x) = 3x-cosx-1 using the secant method, in which the real roots of the equation f(x) lies in the interval (0, 1).

Solution:

Using the given data, we have,
x0 = 0 , x1 = 1 and
f(x0) = f(0) = 3x-cosx-1 = -2 ,
f(x1) = f(1) = 3x-cosx-1 = 1.46
Using the secant method formula, we can write

The first approximation is:

x1+1 = x1 - [(x1 - x1-1) / f(x1) - f(x1-1)] * f(x1)
x2 = x1 - [(x1 - x0 )/ f(x1) - f(x0)] * f(x1)
x2 = 1 -[(1-0) /1.46-(-2)]*(1.46)
x2 = 0.58

The second approximation is:

x3 = x2 - [(x2 - x1 )/ f(x2) - f(x1)] * f(x2)
x3 = 0.58 -[(0.58-1) /(-0.10)-1.46]*(-0.10)
x3 = 0.64

The third approximation is:

x4 = x3 - [(x3 - x2 )/ f(x3) - f(x2)] * f(x3)
x4 = 0.64 -[(0.64-0.58) /0.12-(-0.10)]*(0.12)
x4 = 0.57

The fourth approximation is:

x5 = x4 - [(x4 - x3 )/ f(x4) - f(x3)] * f(x4)
x5 = 0.57 -[(0.57-0.64) /(-0.13)-0.12]*(-0.13)
x5 = 0.61

The fifth approximation is:

x6 = x5 - [(x5 - x4 )/ f(x5) - f(x4)] * f(x5)
x6 = 0.61 -[(0.61-0.57) /0.01-(-0.13)]*(0.01)
x6 = 0.61
Since, x5 and x6 matching up to two decimal places, the required root is 0.61
